# libpythonpro
Módulo para exemplificar contrução de projetos Python no curso PyTools

Nesse curso é ensinado como constribuir com projetos de código aberto 

link para o curso [PythonPro](https://www.python.pro.br/)

suportado versão 3 de Python

Para instalar:
'''Console
python3 -m venv .venv
source .venv/bin/activate
pip install -5 requiments-dev.txt'''

Para confirir a qualidade do código:
'''Console
flake8
'''

Tópicos a serem abordados:
1. Git
2. Virtualenv
3. pip
